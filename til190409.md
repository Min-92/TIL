    - 기존에 풀어본 형식이지만 C++으로는 처음
    - 속도맞추기가 꽤 까다로웠음
    - 해결 키워드는 boolean 타입 vector
    ~~~
    vector <bool> A(N+1,true);
    for(int i = 2; i <= N; i++){
        if(A[i] != false){
        answer += i;
        }
        for(int j = 2; j*i <= N; j++){
            A[i*j] = false;
        }
    }
    ~~~

4. 2-1 백준 1912 연속합
    - 오늘의 문제
    - 조건만 잘 따지면 쉬운문제지만 제대로 못잡고 헤맴
    - 반복문 두개를 썼더니 시간초과
    - 결국 반복문 하나 + 조건 수정으로 해결
~~~
    int sum = 0;
    int max = arr[0];

    for(int i = 0; i < n; i++){
        sum = sum+arr[i];
        if(arr[i] > sum){
            sum = arr[i];
        }
        if(sum > max){
            max = sum;
        }
    }
~~~
